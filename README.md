# IPL Database Analysis using SQL

## Project Overview
This project analyzes IPL cricket data using SQL.

## Database Tables
- Teams
- Players
- Matches
- Scores

## Features:
- Team Wise Total Runs
- Top 5 Run Scorers
- Most Sixes Hit Player
- Most Fours Hit Player
- Average Runs Per Player
- Match Wise Highest Score

## Tools Used
- MySQL

## SQL Concepts Used
- JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions
- Foreign Keys
